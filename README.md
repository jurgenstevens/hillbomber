# Hillbomber
Hillbomber is a skateboarding game based in San Francisco, California where the user skates down hills at a high speed dodging obstacles along the way.

[Click Here To Play Hillbomber!](https://jurgenstevens.github.io/hillbomber/)

# Game Instructions:
1) In this game, the player must dodge the following obstacles: cars, dogs and pot holes.

2) Player must dodge the obstacles or they will wipe out and the game will be over.

3) The score will increase as time progresses.

4) The game will require the following buttons to move the skateboard:
	"w" for up, "a" for left, "s" for down, and "d" for right.

5) Once the score reaches 10, the player wins.
